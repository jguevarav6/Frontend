  INSTALACIÓN Y USO
═══════════════════════════════════════════════════════════════════

🚀 INSTALACIÓN COMPLETA
────────────────────────────────────────────────────────────────────

1. Instalar todas las dependencias automáticamente:

   npm install

   Esto lee package.json e instala:
   - Todas las dependencies (producción)
   - Todas las devDependencies (desarrollo)


2. Instalar solo dependencias de producción:

   npm install --production

   Solo instala 'dependencies', omite 'devDependencies'


🔧 INSTALACIÓN MANUAL (por paquete)
────────────────────────────────────────────────────────────────────

Si necesitas instalar paquetes individuales:

# Angular Core
npm install @angular/core@~12.2.0 @angular/common@~12.2.0

# NGRX
npm install @ngrx/store@^12.5.1 @ngrx/effects@^12.5.1

# jsPDF
npm install jspdf@^2.5.3 jspdf-autotable@^3.8.5

# RxJS
npm install rxjs@~6.6.0


📋 VERIFICAR VERSIONES INSTALADAS
────────────────────────────────────────────────────────────────────

# Ver todas las dependencias con sus versiones
npm list --depth=0

# Ver versión de un paquete específico
npm list @angular/core


🔄 ACTUALIZAR DEPENDENCIAS
────────────────────────────────────────────────────────────────────

# Ver paquetes desactualizados
npm outdated

# Actualizar a versiones compatibles (respeta ^ y ~)
npm update

# Actualizar Angular a versión específica
ng update @angular/core@12 @angular/cli@12

  NOTAS IMPORTANTES
═══════════════════════════════════════════════════════════════════

⚠️  COMPATIBILIDAD DE VERSIONES
────────────────────────────────────────────────────────────────────

- Angular 12 requiere Node.js >= 12.20, 14.15, o 16.10
- TypeScript 4.3.x es compatible con Angular 12.2.x
- NGRX 12.x es compatible con Angular 12.x
- RxJS 6.6.x es compatible con Angular 12.x


📌 SÍMBOLOS EN VERSIONES (package.json)
────────────────────────────────────────────────────────────────────

- "~12.2.0"  → Permite actualizaciones de parches (12.2.1, 12.2.2)
- "^12.5.1"  → Permite actualizaciones menores (12.6.0, 12.7.0)
- "12.2.0"   → Versión exacta, sin actualizaciones automáticas


🎯 PAQUETES CLAVE DEL PROYECTO
────────────────────────────────────────────────────────────────────

1. @ngrx/store + @ngrx/effects
   → Gestión de estado global tipo Redux
   → Usado en: src/app/features/items/state/

2. jspdf + jspdf-autotable
   → Exportación de reportes en PDF
   → Usado en: items-list.component.ts (método exportToPDF)

3. rxjs
   → Observables para datos asíncronos
   → Usado en: Todos los servicios y componentes

4. @angular/router
   → Navegación entre páginas
   → Usado en: app-routing.module.ts
